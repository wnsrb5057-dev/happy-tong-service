# 해피통서비스 수동 테스트 체크리스트

## 로그인

- checker / 1234 로그인 가능
- admin / 1234 로그인 가능
- 로그아웃 가능
- 잘못된 계정 입력 시 오류 표시

---

## 체커 화면

### 홈

- 오늘 확인 대상자 표시
- 대상자 카드 클릭 가능
- 이상징후 보고 바로가기 가능

### 대상자

- 배정 대상자 목록 표시
- 대상자 상세 진입 가능
- 위험도 배지 표시

### 기록작성

- 대상자 선택 가능
- 확인 유형 선택 가능
- 기록 저장 가능
- 저장 후 이력에 반영

### 이력

- 본인 작성 기록 표시
- 이상징후 보고 이력 표시

---

## 관리자 화면

### 대시보드

- 오늘 운영 현황 표시
- 확인 예정/완료/미작성 수 표시
- 이상징후 요약 표시

### 이상징후

- 이상징후 목록 표시
- 필터 동작
- 상세보기 가능
- 처리 상태 변경 가능

### 대상자

- 대상자 목록 표시
- 위험도 필터 동작
- 상세보기 가능
- PC/모바일 카드 레이아웃 확인

### 체커

- 체커 목록 표시
- 담당 대상자/활동 현황 표시

### 확인기록

- 기록 목록 표시
- 필터 동작

### 통계

- 통계 카드 표시
- 차트 레이아웃 깨짐 없음

### 보고서

- 기간 선택 가능
- 보고서 초안 생성 가능
- 미리보기 화면 이동 가능
- PDF 내보내기 가능
